# Some Optimization basic stuffs:


1. Basic Optimization:<br>
1.1. Linear Optimization: (https://web.stanford.edu/~boyd/cvxbook/bv_cvxbook.pdf#page=21.54)
<img width="745" height="130" alt="image" src="https://github.com/user-attachments/assets/816f16bd-0ea0-4339-9608-e77100598344" /> <br>
  in which:<br>
  - $x$: decision variables  <br>
  - $c$: cost vector, mỗi $c_j$ cho biết biến $x_j$ đóng góp bao nhiêu vào objective <br>
  - $a_i$: $i_{th}$ constraint vector  <br>

    Objective:<br>
    $c^Tx = c_1x_1 + c_2x_2 + \cdots + c_nx_n$<br>
    Constraint:<br>
    $a_i^Tx \le b_i$ <br>
    Expanded:<br>
    $a_{i1}x_1 + a_{i2}x_2 + \cdots + a_{in}x_n \le b_i$<br>
